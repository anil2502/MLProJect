# MLProject - Student Exam Performance Indicator

A complete end-to-end Machine Learning project focused on predicting student exam performance using data preprocessing, feature engineering, exploratory data analysis, and predictive modeling techniques.

This project demonstrates how Machine Learning can be used to analyze student-related factors and predict academic performance using modern ML workflows and deployment-ready architecture.
 
---

# Features

- Student Exam Performance Prediction
- End-to-End ML Pipeline
- Data Preprocessing
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Model Training & Evaluation
- Multiple ML Algorithms
- Pipeline Serialization
- Modular Project Structure
- Deployment-ready Architecture

---

# Problem Statement

The goal of this project is to predict student exam scores based on several influencing factors such as:

- Gender
- Race/Ethnicity
- Parental Education
- Lunch Type
- Test Preparation Course

Using these features, the model predicts student performance and helps analyze educational trends.

# Installation

## 1. Clone Repository

```bash
git clone https://github.com/anil2502/MLProJect.git
```

---

## 2. Navigate to Project

```bash
cd MLProJect
```

---

## 3. Create Virtual Environment

### Windows

```bash
python -m venv ml-env
ml-env\Scripts\activate
```

### Linux / Mac

```bash
python3 -m venv ml-env
source ml-env/bin/activate
```
